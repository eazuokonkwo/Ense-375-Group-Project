# Student Grade Analyzer

## Course
**ENSE 375 – Software Testing and Validation**

## Project Overview
The Student Grade Analyzer is a software system designed to:

- Analyze student grade data  
- Compute academic performance metrics  
- Ensure correctness through systematic software testing techniques  

The project addresses inefficiencies in manual grade management, supporting accurate and consistent evaluation of student performance.

## Repository Structure
- `REPORT.md` – Main project report containing the problem definition, team meetings, and project management details
- `assets/` – Images and supporting files (e.g., university logo)

## Current Status
This repository currently contains the initial project documentation required for the early project milestone. Design, implementation, and testing artifacts will be added incrementally as the project progresses.
